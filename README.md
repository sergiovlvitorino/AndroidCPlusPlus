# AndroidCPlusPlus
This code has the intention to show how to use C++ into a Android application.  

Tools:
  Eclipse ADT
  Android SDK 10(2.3.3)
  NDK r9d

Steps:
  Download the source;
  Using a terminal, launch the command lines:
    $  cd AndroidCPlusPlus/jni
    $  ndk-build -B -j4
  Import the project to Eclipse and run it;
